Accessing data with our API

About the API

If you have programming knowledge you can use this API to access datasets in the Use land and property data service. This is a RESTful API in JSON format, making it suitable for a range of programming languages.

What you can access with the API

You can use the API to get:

  • a list of HM Land Registry datasets that are available from this service
  • a list of HM Land Registry datasets that are not part of the service
  • information and metadata about datasets
  • secure links for downloading datasets

You can access datasets that show:

  • UK companies that own property in England and Wales
  • Overseas companies that own property in England and Wales
  • National Polygon Service

How to access the datasets

You need to create an account to agree to a licence to access the datasets.

  1. Create an account if you do not have one
  2. Sign in to your account
  3. Agree to the data licence(s) on the Use land and property data website
  4. Get your API key
