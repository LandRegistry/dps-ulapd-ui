The dataset

The National Polygon Service contains 3 licensed, chargeable datasets:

  • National Polygon dataset
  • Title Descriptor dataset
  • Title Number and UPRN Look Up dataset

These datasets are only available as part of the National Polygon Service and you'll need to agree to a licence to use them.

National Polygon dataset

This dataset shows the indicative shape and position of each boundary of a registered title for land and property in England and Wales. Every freehold or leasehold title has at least one index polygon.

This dataset has more than 25 million titles and 28 million polygons. HM Land Registry index polygons are mapped against MasterMap, Ordnance Survey's large scale map.

Included in this dataset

The data includes:

  • shape
  • polygon identification
  • title number
  • polygon creation date
  • update date - date when title was changed
  • version of polygon identification number
  • record status - shows changes to the record

Read the technical specification for the National Polygon dataset

Title Descriptor dataset

This dataset describes legal interests that are recorded against freehold and leasehold estates registered in England and Wales. (An estate is property and/or land.) An estate is given a unique title number when it is registered. This means you can get data from individual records.

Included in this dataset

The data includes:

  • title number
  • estate interest
  • class of title
  • registered status
  • record status

Read the technical specification for the Title Descriptor dataset

Title Number and UPRN Look Up dataset

This dataset contains HM Land Registry title numbers and Unique Property Reference Numbers (UPRN) for freehold and leasehold land and property registered in England and Wales.

UPRNs are unique 12 digit numbers created by local authorities and Ordnance Survey. Every addressable location in the UK gets a UPRN for the lifetime of the property. Ordnance Survey uses AddressBase to link property information to a map whenever possible.

Title number and UPRNs can only be linked if there is relationship between Ordance Survey's AddressBase address and the title number.

Using UPRNs and title numbers together allows you to match and merge datasets from different sources.

Included in this dataset

The data includes:

  • title number
  • UPRN

Read the technical specification for the Title Number and UPRN Look Up dataset

Buy the dataset

Contact us if you want to buy this dataset. It costs £20,000 per year plus VAT.

You'll be able to access the data after you've:

  • agreed to licences from Ordnance Survey and HM Land Registry
  • paid for the dataset

Dataset licence

Your use of the National Polygon Service is governed by the terms and conditions of a licence.

We use data from Ordnance Survey to prepare polygons and the dataset. This data including the UPRN and polygon ID is under licence.

You'll need to contact Ordnance Survey and enter into a licence agreement with them to use this data.

We can only supply this dataset to you once you've entered into this agreement.




Access the dataset

Download the data

The dataset is updated on the second working day of the month. The data includes an ESRI shapefile and CSV files. You can download the:

  • complete file - this is a complete file of the current data and contains all of the changes that happened during the month
  • change only file - this only includes data that has changed since the previous month's release
  • validation file - this checks that the data has been downloaded correctly

The Title Number and UPRN Look Up dataset does not have a change only file.

Access the data with an API

You'll get an API key when you've:

  • agreed to the licences
  • paid for the data
  • signed in to your account

For more information, read our guidelines on accessing data with the API and API technical details.
